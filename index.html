Personal Website

This project was all about building my own personal portfolio website. The project was completed using only HTML5 and CSS3. It was designed and coded with mobile first principles. It does have a media query to change and adjust the content to fit a different sized viewport. For keeping up to date and for peer reviews I used Git and GitHub to share the progress of the project.
